All About Notes Sharing

“Note Sharing System” is a system used for sharing notes using the website. Notes that the user wants for a particular subject can be found in the website. This system is developed using Python Django and sqlite. Firstly, the user need to sign up and then, they can log in to the system.

Notes Sharing System is a open source you can Download and edit notes as per you need.
